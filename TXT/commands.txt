start - Start the bot
keyboard - Show keyboard
help - Show help instructions
settings - All commands of bot
clean - Delete cookies & broken downloads
search - Find videos directly in bot
cookie - Download cookie from web [youtube/tiktok/x/twitter/custom]
cookies_from_browser - Get cookies from embedded browser
playlist - Show how to download playlists
audio - + URL - Download only audio (mp3)
format - Change quality & format options
mediainfo - Turn ON/OFF sending mediainfo
split - Change splitted video part size (100MB-2GB)
subs - Enable/disable subtitle embedding for videos
link - + [quality] URL - Get direct video/audio stream links for media players
proxy - on/off - Enable/disable proxy for all yt-dlp downloads
check_cookie - Check and validate cookie file
save_as_cookie - Save text as cookie (or upload TXT-doc)
tags - Get all your #tags
usage - Get your logs
